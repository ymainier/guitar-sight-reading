This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Commands

- `pnpm dev` - Start development server
- `pnpm build` - Type-check with TypeScript and build for production
- `pnpm lint` - Run ESLint
- `pnpm preview` - Preview production build

## Tech Stack

- React 19 with TypeScript
- Vite 7 for bundling
- VexFlow for music notation rendering
- ESLint with typescript-eslint, react-hooks, and react-refresh plugins

## Project Overview

A guitar sight-reading trainer that displays musical notes on a staff and presents 4 tab notation options for the user to choose from. The app follows a minimalist black-and-white design aesthetic (shadcn/ui style).
