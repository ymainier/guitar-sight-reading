# Guitar Sight Reading

A minimalist guitar sight-reading trainer that helps you learn to read standard notation and connect it to guitar tablature.

## Core Features

Interactive Staff Display - Shows musical notes on a standard staff using VexFlow
Answers - Displays 4 proposed tab notation underneath, the user should select one
Progressive Difficulty - Start with simple notes and gradually increase range/complexity
Clean, Minimal Design - Black and white color scheme with shadcn/ui like components for a focused learning experience

## User Flow

Simple interface with the staff/tab display front and center
Optional settings for note range (e.g., first 5 frets, open strings only, etc.)
Visual feedback when revealing answers
